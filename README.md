# CYPRIOCOIN 2020

## Description
This is a block-chain application that will be used to make the Cyprio-coin.

## Technologies used.

* Postman App.
* Git bash.
* Atom IDE.

## Development
Run all tests:

```
$ npm install
$ npm test
$ npm run dev
```
or on Windows...
```
$ npm install
$ npm run test-windows
$ npm run dev
```

## Bugs.
The chain string is not being identified by websockets.
